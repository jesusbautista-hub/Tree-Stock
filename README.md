# 🌳 Proyecto Tree-Stock - Sistema de Inventario

## Objetivo
Este proyecto implementa un **Árbol Binario de Búsqueda (ABB)** en Java para gestionar un inventario de productos de forma eficiente y ordenada.

## Instrucciones de Ejecución
1. Descargue los archivos `.java`.
2. Compile con: `javac *.java`
3. Ejecute con: `java Main`
